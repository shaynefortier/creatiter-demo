Note: I have experience in web design in the past, so I make the website to be responsive to different devices we use. You can test it by resizing the window. I learned new stuff from web programing conferences, so I started to apply them such as CSS Grid, CSS Flexbox, CSS selectors and CSS variables. The validator with show errors, because I use the variable, but they are compatible to all modern browsers. Otherwise, there is no error.

I tested and works on Google Chrome, Firefox and Microsoft Edge. Internet Explorer is not compatible with CSS Grid and variables, so the website looks bad. According to Statcounter, the visitors using Internet Explorer represents 3.15% this year.
